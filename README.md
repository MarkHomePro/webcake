
# Webcake-Chanon

เว็บไซต์นี้ถูกสร้างขึ้นเพื่อการศึกษา เนื้อหาเกี่ยวกับเค้กในเว็บนี้ รวมถึงข้อมูลและราคา เป็นเพียงข้อมูลสมมุติเท่านั้น ไม่มีการซื้อขายจริง

## ผู้จัดทำ

1.  นายชานนท์ ปฏิพิมพาคม
2.  นายนาชน ปฏิพิมพาคม
